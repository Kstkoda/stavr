# github.read_file

Read a file from a repo at a given ref. Returns decoded text content + metadata. Binary files surface as base64 in `raw_base64`.

## Tier behaviour

AUTO — runs without confirmation. Read-only or has built-in escape hatches; the caller is responsible for using it intentionally.

## Input schema

```json
{
  "type": "object",
  "properties": {
    "repo": {
      "type": "string",
      "minLength": 1
    },
    "path": {
      "type": "string",
      "minLength": 1
    },
    "ref": {
      "type": "string"
    }
  },
  "required": [
    "repo",
    "path"
  ],
  "additionalProperties": false
}
```
