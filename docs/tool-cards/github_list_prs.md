# github.list_prs

List pull requests for a repo. Returns summaries (number, title, state, author, head/base, updatedAt).

## Tier behaviour

AUTO — runs without confirmation. Read-only or has built-in escape hatches; the caller is responsible for using it intentionally.

```json
{
  "type": "object",
  "properties": {
    "repo": {
      "type": "string",
      "minLength": 1
    },
    "state": {
      "type": "string",
      "enum": [
        "open",
        "closed",
        "merged",
        "all"
      ],
      "default": "open"
    },
    "limit": {
      "type": "integer",
      "minimum": 1,
      "maximum": 200,
      "default": 30

  "required": [
    "repo"
  ],
  "additionalProperties": false
}
```
