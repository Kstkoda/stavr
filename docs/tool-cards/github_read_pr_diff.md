# github.read_pr_diff

Unified diff for a PR as plain text. Truncated to 1 MiB; if truncated, `truncated: true` and `bytes_truncated` are set.

## Tier behaviour

AUTO — runs without confirmation. Read-only or has built-in escape hatches; the caller is responsible for using it intentionally.

```json
{
  "type": "object",
  "properties": {
    "repo": {
      "type": "string",

      "minimum": 1
    }
  },
  "required": [
    "repo",

  "additionalProperties": false
}
```

## Output schema

```json
{
  "type": "object",
  "properties": {
```
