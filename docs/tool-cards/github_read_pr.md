# github.read_pr

Read a GitHub pull request by repo + number. Returns title, body, state, head/base refs, CI status summary, files changed, mergeability.

## Tier behaviour

AUTO — runs without confirmation. Read-only or has built-in escape hatches; the caller is responsible for using it intentionally.

## Input schema

```json
{
  "type": "object",
  "properties": {
    "repo": {
      "type": "string",
      "minLength": 1
    },
    "number": {
      "type": "integer",
      "minimum": 1
    }
  },
  "required": [
    "repo",
    "number"
  ],
  "additionalProperties": false
}
```

## Output schema

```json
{
  "type": "object",
  "additionalProperties": {}
}
```

## Side effects

- shells out to `gh pr view --json` (read-only)


## Example

```json
{
  "args": {
    "repo": "Kstkoda/cowire",
    "number": 9
  },
  "result": {
    "number": 9,
    "title": "Spec 40 Phase 2",
    "state": "MERGED"
  }
}
```

## Error modes

- gh CLI not authenticated or repo not visible → `{ code: "gh_failed", ... }`
